# Keyboard Shortcuts

## Application Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd+H` | Toggle sidebar (host list) |
| `Ctrl/Cmd+,` | Toggle settings panel |
| `F1` | Toggle shortcuts help |
| `F3` or `Alt+D` | Open debug window |

## Tab Management

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd+T` | New tab (same host) |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Ctrl/Cmd+W` | Close current tab |

## Terminal

| Shortcut | Action |
|----------|--------|
| `Shift+PageUp` | Scroll up |
| `Shift+PageDown` | Scroll down |
| `Ctrl/Cmd++` | Zoom in |
| `Ctrl/Cmd+-` | Zoom out |
| `Ctrl/Cmd+0` | Reset zoom |

## Text Selection

| Shortcut | Action |
|----------|--------|
| `Shift+Click+Drag` | Select text (default mode) |
| `Alt/Option+Drag` | Block (rectangular) selection |
| `Double-click` | Select word |
| `Triple-click` | Select line |

## Clipboard

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+C` | Copy selected text |
| `Ctrl+Shift+V` | Paste from clipboard |
| `Shift+Insert` | Paste from clipboard (alternative) |

> **Note:** Selection is automatically copied to clipboard when in "Auto-copy" mode.

## File Manager (SFTP)

Open file manager by clicking the folder icon next to a host in the sidebar.

| Shortcut | Action |
|----------|--------|
| `Tab` | Switch between panes |
| `Enter` | Open directory / Download file |
| `F5` | Copy files |
| `F6` | Move files |
| `F7` | Create directory |
| `F8` or `Del` | Delete files |
| `F2` | Rename |
| `Space` | Select/deselect file |
| `Insert` | Select and move to next |
| `Ctrl/Cmd+A` | Select all |
| `Escape` | Close file manager |

## Emoji

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+E` | Open emoji picker |
